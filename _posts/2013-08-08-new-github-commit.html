BootstrapPHP-imdbSearch: a demo built of that uses the IMDB API from mymovieapi.com. This code is using PHP5 and Simple XML as the backend. The front end is created using bootstrap. The folder structure on your server should be:

Web Root

- index.php
- imdb.php
/css
/fonts

In other words, move files out of the app folder.

View repo: https://github.com/jennifert/Bootstrap-imdbSearch

View demo site: http://jenntesolin.com/demos/Bootstrap-IMDB/index.php
