BootstrapPHP-imdbSearch: a demo built of that uses the IMDB API from omdbapi.com/. This code is using PHP5 and Simple XML as the back-end. The front end is created using bootstrap. The folder structure on your server should be:

Web Root

- index.php
- imdb.php
/css
/fonts

In other words, move files out of the app folder.

View repo: https://github.com/jennifert/Bootstrap-imdbSearch

View demo site: http://bootstrap-imdbsearch.azurewebsites.net/
demos/
